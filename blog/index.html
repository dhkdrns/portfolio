Complex CSS selectors

As our documents grow, we'll need to leverage more complex CSS selection syntax. Here are a few important selectors to cherish and to keep:

1. The Universal Selector

* {

//css that applies to every element

}

2.The Class Selector

div.container {//css that only applies to divs with a class of 'container' }

3.The Child Selector

div>p { //css that only applies to direct child elements (no grand-children) of an element //in this case, all of the child paragraph elements of divs across the page }

4.The Decendant Selector

div p { //css that applies to all descendants of an element of a certain type //in this case, all paragraph elements of divs across the page (even if they're nested in //other elements, like spans or lists) }
